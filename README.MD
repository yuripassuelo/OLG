

# Exemplos de gerações sobrepostas

Exemplo baseado no modelo seminal de gerações sobrepostas com riscos idiossincráticos de Hugget (1996)

- Julia
- Matlab

A versão relacionada a Matlab faz parte de um trabalho de disciplina e possui uma estrutura mais complexa, uma vez que tanto a estrutura demográfica, com a taxa de crescimento populacional $\eta$ quando o vetor de sobrevivência $\Gamma_t$ são alterados.

A versão em Julia corresponde a uma implementação mais simples e suscinta
